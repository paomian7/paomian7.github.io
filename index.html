重拾

发表于 2018-06-24

写在前面: 一年半之前，使用Hexo和Github Pages搭建了自己的个人博客，没有坚持去写(也不知道去写什么)。今天重拾，望用写作来静下心思考。记录技术、生活、厨艺上的改变。

为什么要有自己的独立博客

目的不在于有多大的关注度，所以不用操心写的好不好。博客主要会写些技术文章，总结并去思考，日后也可以去温故知新。
现在BSP(Blog Service Provider)很多，有了账号就可以去写东西，那为什么现在还要自己去搞个博客？如果用别的的东西，你要忍受上面的广告等等。并且独立的博客就是你自己的，可以自己决定用什么域名、主题、字号、字体，所有都由自己决定。
一言以蔽之，独立的才是自己的

为什么是GitHub和Hexo

GitHub的伟大想必所有开发者都知道，虽然已被Microsoft收购。
GitHub Pages有以下优点：

  • 免费、服务稳定
  • 可以绑定自己的域名(后续会绑定)
  • 搭建的博客是轻量级的
  • 可以使用Markdown标记语言(Mac下我用的UIysses很好用)
    关于Hexo大家可以google，怎么去和GitHub搭配使用，这里就不贴教程了

下一篇技术文章准备中，先这样，该去健身了。
